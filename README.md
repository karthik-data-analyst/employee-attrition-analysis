# 📊 Employee Attrition Analysis & Prediction

## 📌 Project Overview
This project analyzes employee attrition using SQL, Python, and Power BI to identify key drivers of employee turnover and predict high-risk employees.

---

## 🛠 Tools Used
- SQL (MySQL)
- Python (Pandas, NumPy, Scikit-learn)
- Power BI
- Excel

---

## 🔍 Analysis Performed
- Cleaned and processed 10,000+ employee records
- Conducted Exploratory Data Analysis (EDA)
- Calculated department-wise attrition rates
- Built Logistic Regression model (80% accuracy)

---

## 🤖 Model Performance
Accuracy: **80%**
